# Базовые правила

## Введение в игру

Варгейм "Викинги" - стратегическая настольная игра для 2-4 игроков, где каждый игрок управляет армией одной из фракций эпохи викингов.

## Необходимые компоненты

Для игры вам понадобится:
- Игровое поле
- 10-20 фишек для каждой армии
- 6-гранный кубик (d6)

## Настройка игры

1. Разместите игровое поле на столе
2. Каждый игрок выбирает фракцию
3. Расставьте стартовые юниты согласно сценарию
4. Определите первого игрока броском кубика

## Структура хода

### Фазы хода:

| Фаза | Описание |
|------|----------|
| **Планирование** | Игрок выбирает действия для юнитов |
| **Движение** | Перемещение юнитов по полю |
| **Бой** | Проведение атак |
| **Мораль** | Проверка боевого духа |

## Механика бросков

### Бросок на атаку:
Результат = Бросок 2х кубиков d6 + Модификатор атаки от фракции

## Характеристики юнитов

Каждый юнит имеет 4 основные характеристики:

| Характеристика | Описание |
|----------------|----------|
| **Атака (АТК)** | Сила в ближнем бою |
| **Защита (ЗЩТ)** | Способность выдерживать урон |
| **Скорость (СКР)** | Количество клеток за ход |
| **Здоровье (ЗДР)** | Количество попаданий |

## Условия победы

Победа достигается одним из способов:

1. **Полное уничтожение** всех вражеских юнитов
2. **Достижение целей сценария**
3. **Сдача** противника

## Советы для новичков

1. **Не распыляйте силы** - держите юниты группами
2. **Используйте рельеф** - высота дает преимущество
3. **Контролируйте фланги** - это ключ к победе
4. **Экономьте действия** - иногда оборона лучше атаки
